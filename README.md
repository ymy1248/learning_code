# 学习代码

## 概括
写这个主要是因为自己看的技术书总是没办法有深刻的印象，还是需要有用代码例子来加深自己的认识。并且希望自己同时能把工作中遇到的代码问题也记录下来。包括以下内容：
- **Effective C++**: 以代码，注释和笔记的形式学习。
- **Effective modern C++**: 以代码，注释和笔记的形式学习。
- **APUE**：以代码注释的方式学习。
